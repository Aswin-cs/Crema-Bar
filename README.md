# CREMA BAR

CREMA BAR is a premium, state-of-the-art Next.js showcase web application designed for a demo cafe/dessert shop. This project serves as a high-fidelity interactive prototype for showcasing web design and development capabilities to potential clients.

## Key Features

- **Dynamic Hero Section**: Includes an auto-looping marquee of signature cream desserts and dishes.
- **Interactive Menu**: Fully detailed showcase of premium items with high-quality imagery.
- **Custom Animations**: Leveraging `framer-motion` and `gsap` for elegant scroll-driven details and micro-animations.
- **Bento Grid Media Layout**: Dynamic showcase grid matching modern aesthetic trends.
- **Custom Interactive Cursor**: Enhanced user experience with a floating indicator.
- **Full Responsiveness**: Optimized for mobile, tablet, and desktop viewing.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to view the application.

## Technologies Used

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **Icons**: React Icons
- **Fonts**: Montserrat (Branding), Raleway (UI Text)
